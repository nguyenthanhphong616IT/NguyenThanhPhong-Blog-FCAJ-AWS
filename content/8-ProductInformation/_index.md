### Product Information: SmartCV (AI-CV AWS)

**SmartCV** is an AI-powered application designed to help users create, manage, and optimize their resumes intelligently. Built and deployed on AWS infrastructure, it leverages modern cloud technologies to deliver a seamless experience.

* **Live Demo / Project Link:** [https://smartcvknight.click/](https://smartcvknight.click/)
* **Source Code (GitHub):** [https://github.com/huynhnhan68/smartcv](https://github.com/huynhnhan68/smartcv)

#### Key Highlights
- Integrated with AI to analyze and improve CV content.
- Fully deployed on AWS using scalable and reliable cloud services.
